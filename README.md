# Aman R Khanchandani Portfolio

Recruiter-facing portfolio for Applied AI + Product Engineering work: Stateframe, PayRail, BidMyTime, AI Systems Notes, open-source contributions, and technical writing.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Vercel

## Local Development

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:4177` when running with:

```bash
npm run dev -- -p 4177
```

## Checks

```bash
npm run lint
npm run build
```

## Deployment

The repo is linked to Vercel. Production deploy:

```bash
npx vercel --prod --yes
```

## Status

Active portfolio surface. The skills section intentionally separates defended strengths from tools currently being built through projects.
