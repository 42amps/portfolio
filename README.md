# Portfolio

Recruiter-facing portfolio for applied AI and product engineering work: Stateframe, PayRail, AI Systems Notes, and selected projects.

## Local Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Lint

```bash
npm run lint
```

## Deployment

This is a static Vite site. It can be deployed to Vercel, Netlify, Cloudflare Pages, or GitHub Pages.

Typical Vercel settings:

- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`

## Project Structure

- `src/main.tsx` - portfolio content and sections.
- `src/styles.css` - Tailwind entrypoint and custom layout styles.
- `index.html` - document shell and SEO description.
